# /dev-dev — Development Server

## Objectif
Lancer le serveur de développement Next.js avec Turbopack.

## Commande

```bash
pnpm dev
```

## Accès
- Application : `http://localhost:4105`
- Prisma Studio : `http://localhost:5555` (si lancé séparément)

## Quand l'utiliser
- Développement quotidien
- Tests manuels
- Hot reload

```

Status : ✅ Lancé

Turbopack : activé

```
