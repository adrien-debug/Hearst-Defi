# Table

Compound table primitive with 4 variants, 3 sizes, sticky header, sortable headers.

```tsx
<Table variant="striped" stickyHeader>
  <Thead>
    <Tr>
      <Th sortable direction="desc">APY</Th>
      <Th>Asset</Th>
    </Tr>
  </Thead>
  <Tbody>
    <Tr><Td numeric>9.4-12.8%</Td><Td>BTC</Td></Tr>
  </Tbody>
</Table>
```
